## Reference Resolution Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                   Reference Resolution                             │
└─────────────────────────────────────────────────────────────────┘

  Spec File
  ┌─────────────────┐
  │ authority:      │
  │   inherits_from: │
  │     - genesis    │
  │     - standard-   │
  │       security   │
  │   depends_on:    │
  │     - domain-     │
  │       catalog    │
  └────────┬─────────┘
           │
           │ parse references
           ▼
  ┌─────────────────────────────────────────┐
  │        Reference Parser                  │
  │  @genesis           → local reference   │
  │  @standard-security → local reference   │
  │  domain-catalog     → local reference    │
  └────────────┬────────────────────────────┘
               │
               │ validate targets exist
               ▼
  ┌─────────────────────────────────────────┐
  │        Reference Validator               │
  │                                          │
  │  ✓ genesis.yaml exists                  │
  │  ✓ security.yaml exists                 │
  │  ✓ catalog.yaml exists                  │
  │                                          │
  │  ─────────────────────────────────────   │
  │                                          │
  │  Traceability Report:                    │
  │  • Inherits: genesis, standard-security │
  │  • Depends on: domain-catalog            │
  │  • Impacted by: 3 specs                 │
  └─────────────────────────────────────────┘
```
